# GOZU - 动漫与二次元谷子定制分享推荐平台

> AI 赋能的"IP × 载体 × 设计"聚合与交易平台

./dev.sh

## 📋 项目概述

### 平台定位

面向动漫与二次元谷子（周边）生态的定制电商与内容分享平台，围绕 IP 进行多载体商品的设计、迁移、推荐与交易。

### 核心价值

- **设计者**：AI 辅助创作、多载体变体、流量分成
- **制造商**：对接创意设计、订单生产管理
- **购买者**：发现喜欢的 IP、定制专属谷子
- **管理员**：内容安全、版权管理、平台运营

## 🎯 待实现需求

### P0 - 核心功能

#### 1. IP / 主题库

- **IP 档案页**：授权信息、官方素材、风格标签、热门谷子组合
- **IP 浏览**：分类筛选、热度排行、新番追踪
- **IP 关联**：IP → 设计作品 → 商品载体 的结构化关系
- **版权标识**：授权类型显示、原创声明、版权到期提醒

#### 2. 商品与载体系统

- **载体类型**：徽章、亚克力立牌、围巾、帽子、手办、挂件、手机壳、抱枕、钥匙扣、马克杯等
- **材质工艺**：刺绣、热转印、UV 印刷、烫金、亚克力切割、金属冲压等
- **SKU 管理**：尺寸、颜色、材质组合、定价策略
- **工艺约束**：根据工艺自动限制颜色数量、纹理精度

#### 3. AI 设计能力增强

- **文生图**：输入 IP、风格、配色关键词，生成设计稿
- **图生图**：已有图案风格迁移
- **跨载体适配**：自动调整安全区域、裁切、构图、分辨率
- **批量变体**：同一主题生成多尺寸、多材质、多风格版本
- **版权风险检测**：相似度分析、侵权预警

#### 4. 推荐系统

- **风格推荐**：基于标签/风格向量
- **内容推荐**：基于 IP、主题、热点、季节
- **个性化推荐**：用户偏好/浏览/收藏行为
- **推荐解释**：给出"为何推荐"的可视化说明

#### 5. 交易与订单

- **购物车**：多商品合并、优惠计算
- **支付集成**：微信/支付宝/银行卡
- **订单流转**：待付款 → 待生产 → 生产中 → 待发货 → 已发货 → 已完成
- **物流追踪**：实时物流信息、预计到达时间

### P1 - 增值功能

#### 6. 设计师收益系统

- **分成机制**：作品销售分成比例设置
- **收益看板**：实时收益、趋势分析、结算历史
- **提现功能**：余额提现、税务处理

#### 7. 设计协作与共创

- **多人协作**：邀请协作者、权限分配
- **版本控制**：历史版本、版本对比、回滚
- **评审流程**：设计稿评审、批注反馈

#### 8. 限量发行与盲盒

- **限量系列**：发行数量限制、编号证书
- **时间窗发售**：预售、限时抢购
- **盲盒机制**：随机抽取、概率展示、保底机制

#### 9. AR/3D 预览

- **3D 展示**：围巾、帽子、立牌 360° 展示
- **AR 试穿**：虚拟试戴/试穿效果
- **实景预览**：将设计放入真实场景

#### 10. 主题活动与榜单

## 🏗️ 业务流程

### 设计发布流程

```
设计者上传/AI生成图案 → Gemini理解分析 → 选择载体适配 → 生成SKU → 审核 → 发布
```

### 购买流程

```
浏览推荐 → 选择商品 → 定制调整 → 加入购物车 → 下单支付 → 等待生产 → 物流配送 → 确认收货
```

### 生产流程

```
制造商接单 → 排产计划 → 生产制造 → 质检 → 打包发货 → 物流上传
```

## 👥 用户场景

### 设计者场景

1. **小白创作**：使用 AI 文生图，输入"初音未来 赛博朋克风格 蓝色调"，生成设计稿
2. **专业设计**：上传原创设计，AI 自动适配多种载体尺寸
3. **收益管理**：查看作品销量、收益趋势，申请提现

### 购买者场景

1. **发现好物**：浏览推荐、搜索喜欢的 IP，发现心仪设计
2. **个性定制**：选择设计+载体，调整位置大小，预览效果
3. **下单追踪**：支付订单，实时查看生产和物流状态

### 制造商场景

1. **接单管理**：查看待处理订单，确认生产排期
2. **生产追踪**：更新生产状态，上传质检照片
3. **模板维护**：管理工艺模板，设置生产能力

### 管理员场景

1. **内容审核**：审核新发布设计，处理违规内容
2. **数据监控**：查看平台运营数据，分析业务指标
3. **用户管理**：处理用户申诉，管理账号状态

## 🎨 设计原则

### 用户体验

- **简洁直观**：核心功能一目了然，操作流程清晰
- **即时反馈**：操作结果实时呈现，加载状态明确
- **容错设计**：防止误操作，支持撤销恢复

### 视觉风格

- **毛玻璃效果**：iOS 风格的磨砂玻璃质感
- **渐变色彩**：品牌色渐变，营造高级感
- **微动效**：恰当的动画反馈，提升体验

### 技术架构

- **前后端分离**：Vue3 + FastAPI
- **响应式设计**：适配桌面端和移动端
- **国际化支持**：中英双语切换
